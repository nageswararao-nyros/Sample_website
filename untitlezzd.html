    Using ! important

    !important takes the first priority to apply styles even file having inline or external css to that text...etc.

    The !important rule is a way make sure your most crucial CSS rules are always applied. It is a way to override the general rule of cascading order. If two styles come into conflict in the same element, and one has the !important rule assigned to it, that one will be used.
